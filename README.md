# How to run the project
From Your Terminal or SHELL

1. **Navigate to project directory**
   ```bash
   cd Group-Project-Fullstack
   ```
2. **Navigate to Backend folder**
    ```bash
   cd server
   ```

3. **Install packages**
   ```bash
   npm install
   ```
4. **Run the server**
    ```bash
   node index.js
   ```
Open a new terminal

5. **Navigate to Frontend folder**
    ```bash
   cd client
   ```
6. **Install packages**
    ```bash
   npm install
   ```
4. **Run the client**
    ```bash
   npm run dev
   ```
